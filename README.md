## 这是个什么样的品牌?

Elysium Hope 成立于2022年,专门做手工定制首饰。他们的理念挺特别——"有意义的珠宝"。不是那种只是好看就完了的装饰品,而是真正能承载记忆和情感的东西。

更有意思的是,他们每卖出一件首饰,都会拿出一部分收益捐给需要帮助的人。买个首饰还能做点好事,这感觉还挺不错的。

他们家的东西都是用925纯银或者14K包金做的。包金这个东西,比镀金耐用多了,含金量是镀金的100倍。日常佩戴不容易褪色,价格又比纯金便宜。对我这种想要点品质但预算又有限的人来说,正合适。
